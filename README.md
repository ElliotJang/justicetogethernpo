# Justice Together Website

This repository contains the source code for the Justice Together website.

## Getting Started

Requirements:

- Node.js and npm (recommend installing via nvm)

Install dependencies and run locally:

```sh
npm install
npm run dev
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build

```sh
npm run build
```
